Website Contents Overview

Webpage Description
Home The homepage features a welcoming banner with a high-resolution background image and overlaid text introducing "Discover Limerick." It includes a responsive navigation bar with links to other pages. A section introduces the website's aim with supporting imagery, and another highlights top attractions with card components, utilizing Bootstrap's grid system for layout. A call-to-action button directs users to learn more, and the footer connects users to social media platforms.
About Us The About Us page describes the mission and vision of the "Discover Limerick" website. Using Bootstrap's grid system, the page presents textual content alongside images, ensuring an engaging and mobile-responsive layout. Custom styling highlights key points, and sections flow naturally for a seamless reading experience. The page combines HTML semantics and Bootstrap's utility classes to enhance accessibility and user experience.
Shop The Shop page provides a visually engaging display of products, featuring digital prints of Irish landmarks and artworks. Products are arranged in a responsive grid layout using Bootstrap, with each item showcased in a card component. Cards include high-quality images, titles, descriptions, prices, and an "Add to Cart" button with Font Awesome icons for easy interaction. Users can navigate through the product offerings and use a "Checkout" button for seamless purchases. The page is complemented by a stylish header, a responsive navbar, and a footer providing contact details and social media links, maintaining consistency with the site's design.
Attractions The Attractions page highlights top destinations in Limerick, Ireland, presented through an engaging and responsive design. Each attraction is displayed in a Bootstrap card component with a high-quality image, title, description, and external "Read More" links for further details. Featured sites include King John's Castle, St. Mary's Cathedral, and Thomond Park, showcasing historical and cultural landmarks. The page integrates a visually appealing layout with a consistent header and footer, offering social media links, contact information, and a cohesive user experience across the website.
Contact Page The "Discover Limerick" webpage serves as the main entry point for exploring Limerick, Ireland, featuring a visually appealing design and responsive layout. It includes a Bootstrap-powered navigation bar with links to key sections: Home, About Us, Contact Us, Attractions, and Cart. The page highlights a modern contact form with validation, a modal for terms and conditions, and seamless user interaction. Social media integration is present via Font Awesome icons, linking to platforms like Facebook, Instagram, and LinkedIn. A stylish footer provides contact details, a logo, and helpful tourist information, ensuring a consistent and user-friendly experience across the site.
